## 🎨 Styling Architecture

### Global Styles (`index.css`)
- Reset all margins/padding
- Global font family
- Body background
- Link styling
- Button defaults
- Custom scrollbar

### App Styles (`App.css`)
- App container background gradient

### Landing Page Styles (`Home.css`) - 1200+ lines
- **Navbar**: Fixed navigation, logo, links, Steam button
- **Hero**: Title animation, subtitle, buttons, glitch effect
- **Features**: 4-column grid with hover effects
- **Gameplay**: Two-column layout with preview box
- **Gallery**: Image grid with hover effects
- **CTA**: Call-to-action section
- **Footer**: Footer links
- **Responsive**: Mobile-first media queries

### Steam Page Styles (`Steam.css`) - 950+ lines
- **Steam Header**: Logo and back button
- **Steam Hero**: Game title and rating display
- **Game Info Panel**: Two-column layout (main + sidebar)
- **Tabs**: Tab navigation with underline effect
- **Description**: Game details with features and requirements
- **Screenshots/Videos**: Responsive grid layouts
- **Purchase Box**: Sticky sidebar with purchase options
- **Stats Box**: Live player statistics
- **Languages Box**: Supported languages info
- **Responsive**: Adaptive layouts for all screen sizes

- Ignores `node_modules/`
